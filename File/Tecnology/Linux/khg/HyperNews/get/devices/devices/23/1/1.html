Well I got tired of my investment lying around, so I took my machine back to the dealer to return it. Of course, worked fine when I got to the store! The 2.1.43 kernel has all the right drivers in it for the TX chipset, although the Iwill on-board Adaptec was only able to acheive 10MHz on the SCSI-3 bus, instead of 20MHz. BTW, the board also worked fine with the 2.0.30 kernel despite all the "unknown PCI device" messages.

So I dragged it back to my office whereupon it immediately stopped working (I have witnesses) so I dragged it back to the dealer... yes, it worked like a champ back in the store again... but I returned it anyway. That's the second Iwill motherboard I've had problems with so I'm staying away from them in the future.
