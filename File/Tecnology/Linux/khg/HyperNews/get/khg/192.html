I have a PPP link between two networks connected by an ISDN line(128kbps).  When I try connection at a low baud rate 38400, the connection takes place.
These are the following steps I did to change to a higher baud rate on  the PPP server:
1. Set the serial comm to 115200 using setserial.
2. Set the tty to 115200 using stty ispeed and ospeed.
3. Configure the ugetty in inittab to work communicate at 115200.
4. Made entry in the gettydefs for 115200
5. Made the changes in ifcfg-ppp0 to work at 115200.
6. Made similar changes to /etc/ppp/options file.
