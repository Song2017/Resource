This is a nightly mirror of the Linux Kernel Hackers' Guide. To post to the KHG, you will need to be connected, but this should lower your connection costs.

This archive is created with the getwww program using the command line arguments:

getwww http://www.redhat.com:8080/HyperNews/get/khg.html \
  -S johnsonm embed frame outline show=all admin Response \
  -D HyperNews/SECURED HyperNews/thread.pl HyperNews/edit-subscribe.pl \
  -i

If you create a mirror of the KHG, I ask that you please update your mirror at least nightly. I don't want any out-of-date material floating around. Thanks!
